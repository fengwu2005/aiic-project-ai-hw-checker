# ImageLab 学生报告

## 实现方法

我把业务逻辑集中在 `final/image_ops.py`。`load_image` 负责读取并统一转换为 RGB，避免灰度图、调色板图和透明图在反色、像素遍历时出现通道不一致。`resize_image` 根据比例计算新尺寸，`crop_image` 校验裁剪框是否在图像范围内，`transform_image` 负责把 CLI 操作分发到固定函数。

## 像素处理

`blur_image` 使用 3x3 均值窗口，对 R、G、B 三个通道分别求平均。`edge_detect` 使用固定 Laplacian 卷积核，响应值可能为负或超过 255，所以用 `_clamp` 限制范围。`median_filter` 收集窗口内每个通道的像素值并取中位数，用于去除孤立噪点。

## 验证方法

我用 2x2 黑白图片验证反色，用 4x4 渐变图验证裁剪区域，用 5x5 中心亮点图验证均值模糊，用 5x5 单点噪声图验证中值滤波，用左右黑白分界图验证边缘检测。非法 `scale=0`、越界裁剪框、偶数滤波窗口都应抛出 `ValueError`。

## 关键贡献和不足

我的关键贡献是把固定函数接口、参数校验和像素级滤波实现完整。当前不足是滤波用 Python 循环，处理大图速度一般。
